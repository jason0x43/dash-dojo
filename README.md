Dojo docset for Dash
====================

This is a Dash docset for the Dojo JavaScript toolkit.

Prerequisites
-------------
The following applications and packages need to be available:

* Python 2.7.x
  * Sphinx
  * Cheetah
  * CherryPy
  * Docutils

Building
--------

To build, clone this repo and run:

```
$ npm install
$ npm run build
```
